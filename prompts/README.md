# Prompts

Reusable prompts organized by topic, audience, or company.

- Use `general/` for prompts not tied to a specific technology.
- Use technology folders such as `laravel/`, `node/`, and `vue/` for stack-specific prompts.
- Use `exams/` for exam or role-specific prompt collections.
- Use `company/` for company-specific prompts and supporting snippets that are still prompt-oriented.

Prefer `prompts/<scope>/<purpose>/` when creating new content.
